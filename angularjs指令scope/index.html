angularjs指令directive的scope

所有的directives 都有自己的 scope 对象。 在默认的情况下，directive 不会创建他们自己的scope. 他们会用他们父对象的scope作为自己的scope. 但是 angularjs 允许改变这种默认行为。

scope: false

使用parent 的scope，默认
例子

scope: true

创建一个新的scope,从parent controller 中的 scope 对象继承，（子scope）。
例子

scope: {}

创建一个新的隔离scope,这个隔离的scope 不从parent scope对象继承
例子

注意

子scope和隔离scope在运行起来很像，但是有区别。
子scope参数的初始化会因为继承的关系，从父scope的同名参数里继承下来
隔离scope因为是一个隔离的（独立的scope），所以即使跟包在外面的controller有同名的参数，也不会继承下来。
