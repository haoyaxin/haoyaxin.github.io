内容管理系统CMS

  • WordPress 一个成熟的 CMS/框架，配合插件可以做任何一种网站。高质量的 WordPress 皮肤是一个巨大的市场，很多自由职业者以 WordPress 相关开发为生。配合类似 WP-API 之类的项目你可以把Wordpress变成一组 REST API 。
  • Drupal 8 在今年发布了。这是一次侧重现代开发最佳实践的重构。使用了 Symfony 2 组件、 Composer 包管理器 和 Twig 模板引擎。成千上万的网站在使用 Drupal，它确实是以内容为主的门户网站的一个很好的选择。

移动应用

Ionic, React Native, Meteor

编辑器和开发工具

Atom, Visual Studio Code, NPM, Git
